#### 조건
**1.** 섭씨 온도를 저장하기 위한 변수를 만든다.

`화씨온도 = (섭씨온도*9/5)+32`
**3.** 섭씨 온도와 화씨 온도를 다음과 같이 출력한다.

#### 실행 결과

```python
# 섭씨 온도가 30도일 때
섭씨온도 : 30
화씨 온도 : 86.0

# 섭씨 온도가 10도일 때
섭씨 온도 : 10
화씨 온도 : 50.0
```
